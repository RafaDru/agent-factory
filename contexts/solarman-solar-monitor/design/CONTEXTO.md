# Design — SOLARMAN Solar Monitor

## Proposito
Criar e manter uma interface intuitiva, moderna e focada em dados para o monitoramento solar.

## Diretrizes Visuais
- **Tema:** Dark mode (fundo #0f172a), alta legibilidade.
- **Gráficos:** Usar cores que facilitem a leitura (verde para geração, azul para consumo).
- **Mobile-first:** O dashboard deve ser responsivo e funcionar bem em celulares.
- **Alertas:** Utilizar tons de alerta (laranja/vermelho) apenas quando necessário (ex: assimetria alta, sujeira nos paineis).

## Acoes Disponiveis
- design_ui: Cria esboço de interface ou fluxos de navegação.
- prototype: Gera HTML/CSS/JS funcional para o dashboard.
- improve_ux: Analisa usabilidade e sugere mudanças.

## Stack Sugerida
- Tailwind CSS
- Chart.js (para gráficos de geração)
- Lucide Icons
