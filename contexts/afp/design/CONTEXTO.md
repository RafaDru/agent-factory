# Design — Agent Factory Platform

## Proposito
Evoluir a interface do dashboard do Agent Factory para ser intuitiva, funcional e tecnicamente rica.

## Diretrizes de Design
- **Foco no Monitoramento**: Sempre exibir qual modelo LLM está rodando e o tempo de execução em tempo real.
- **Interatividade**: Cards de projetos devem ser expansíveis/recolhíveis para gerenciar o espaço visual.
- **Customização**: Implementar Light e Dark mode como prioridade.
- **Feedback visual**: Utilizar badges e estados visuais claros.

## Acoes Disponiveis
- design_ui: Cria esboço de componentes.
- prototype: Gera HTML/CSS (Tailwind/Lucide) para componentes.
- analyze_ux: Analisa tendências e propõe melhorias.

## Stack Sugerida
- Tailwind CSS
- Alpine.js (para interatividade rápida sem build complexo)
- Lucide Icons
